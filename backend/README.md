# Backend - Be the hero

## Stack

- `NodeJS`
- `Express`
- [`KnexJS`](http://knexjs.org/)
- `SQLite 3`
- [`Celebrate`](https://github.com/arb/celebrate)
- [`Cross env`](https://github.com/kentcdodds/cross-env)
- [`Postman`](https://www.postman.com/)


## Executar o projeto

- Instalação de pacotes:
```
npm install
```

- Execução do servidor: 
```
npm start
```

- Execução de testes: 
```
npm test
```
